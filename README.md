<h2>6 Months JavaScript Plan</h2>

<h3>Month 1</h3>
   <h4>Day 1: Learned VAR, LET, CONST variable declaration.</h4>
   <ul>
       <li>VAR: can be declared anywhere/globally, and optional to initialize a value.</li> 
       <li>LET: declared locally, and optional to initialize a value, block scoped.</li>
       <li>CONST: declared block scoped, read-only, and CANNOT be changed.</li>
   </ul>
   <h4>Day 2: Learned 7 Primitive Data types and typeof usage.</h4>
   <ul>
      <li>Boolean: can be True or False.</li>
      <li>null: an intentional blank object value.</li>
      <li>undefined: automatically assigned but no value.</li>
      <li>number: can be an integer or float.</li>
      <li>BigInt: different way to store larger numbers.</li>
      <li>String: sequence of characters or text value.</li>
      <li>Symbol - instances that are unique and immutable.</li>
   </ul>
