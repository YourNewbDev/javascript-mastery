<h2>6 Months JavaScript Plan</h2>

<h3>Month 1</h3>
- Day 1: Learned VAR, LET, CONST variable declaration.
  - VAR: can be declared anywhere/globally, and optional to initialize a value.
  - LET: declared locally, and optional to initialize a value, block scoped.
  - CONST: declared block scoped, read-only, and CANNOT be changed.
